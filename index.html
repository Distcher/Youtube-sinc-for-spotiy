Como este overlay funciona

Arquitetura real de produção

Este aplicativo desktop (Electron) implementa a interface completa, o módulo de autenticação OAuth 2.0 + PKCE do Spotify, a busca e adição de faixas via API oficial, o tratamento de erros e o relatório de sessão — tudo pronto para uso real. O login com o Spotify abre uma janela dentro do próprio app; a troca do código por tokens acontece no processo principal do Electron, então nada disso passa pelo navegador do sistema.

A única peça que o app não pode fazer sozinho é ler o conteúdo da aba do YouTube em outro processo/navegador. Para isso, em produção, este mesmo botão flutuante seria acionado por uma extensão de navegador (Chrome/Edge/Firefox) cujo content script roda dentro de youtube.com, lê o título do vídeo/artista a partir do player e envia o evento para este app — exatamente a função que o botão "Simular reprodução" chama aqui no modo demo.

Cole seu Client ID do Spotify (criado em developer.spotify.com) para testar o fluxo de autenticação e adição de faixas de verdade. Cadastre http://127.0.0.1:8888/callback como Redirect URI nas configurações do seu app no painel do Spotify.
